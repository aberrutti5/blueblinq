  <action>
Create a new file at `src/app/api/invoices/export/route.ts` implementing a GET handler that:

1. Validates session with `getServerSession(authOptions)` — return 401 if missing.
2. Retrieves `userId` from `(session.user as { id: string }).id`.
3. Gets `companyId` via `db.companyMembership.findFirst({ where: { userId, isDefault: true }, select: { companyId: true } })` — return 400 if null.
4. Queries `db.invoice.findMany({ where: { companyId }, orderBy: { invoiceDate: 'desc' }, select: { invoiceNumber, invoiceDate, vendorName, vendorRut, invoiceType, subtotal, totalIva, totalAmount, currency, status } })`.
5. Builds CSV string in memory:
   - Header row: `Numero,Fecha,Proveedor,RUT Proveedor,Tipo,Subtotal,IVA,Total,Moneda,Estado`
   - One row per invoice, values comma-separated, fields with potential commas wrapped in double quotes.
   - Dates formatted as `YYYY-MM-DD` using `.toISOString().split('T')[0]` (handle null with empty string).
   - Decimal fields: call `.toString()` on Prisma Decimal (handle null with empty string).
6. Return `new Response(csvContent, { headers: { 'Content-Type': 'text/csv; charset=utf-8', 'Content-Disposition': 'attachment; filename="facturas.csv"' } })`.

Use `Response` (not `NextResponse`) for the CSV download so headers are set correctly. All imports use `@/` path alias. Error messages in Spanish.
